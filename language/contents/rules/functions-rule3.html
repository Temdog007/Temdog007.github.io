Functions may not take a struct as a parameter

        interface IFoo
        {
            int x;
        }

        struct Foo 
        {
            int x; 
        }

        // void PrintFoo(Foo f) => print(f.x); // wrong
        void PrintFoo(IFoo f) => print(f.x); // right

        var foo = new Foo();
        PrintFoo(foo);

Reasoning

Forcing interfaces to be used allows for functions to have more uses. For example, if another new struct happens to implement that interface, it can be used as a parameter to call that function. 

        interface IFoo
        {
            int x;
        }

        struct Foo : IFoo
        {
            int x;
        }

        struct Foo2 : IFoo
        {
            int x;
            int y;
            string s;
        }

        // Structs - Rule 1
        struct Foo3 
        {
            int x; 
        }

        // Functions - Rule 4
        PrintFoo(Foo(), Foo2(), Foo3());
