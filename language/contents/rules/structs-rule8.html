Structs can be marked as poolable to prevent the need for allocating new objects

        interface IFoo
        {
            string s;
        }

        for(var i = 0; i < 100; ++i)
        {
            var f = default(IFoo); // This will create 100 objects
        }
    
        @poolable
        interface IFoo2
        {
            string s;
        }

        for(var i = 0; i < 100; ++i)
        {
            var f = default(IFoo2); // This will create one object the first time and return that same object the next 99 times
        }

Reasoning

Prevent frequent allocating and de-allocating. Some structures were designed to be created frequently and pooling will save on resources.
