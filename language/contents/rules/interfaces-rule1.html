Interfaces can only have readonly values defined.

        interface IFoo
        {
            int x;
        }

        var ifoo = default(IFoo);
        ifoo.x = 5; // error. cannot change value
        ifoo = default(IFoo); // error. cannot reassign

        void ChangeData(IFoo f)
        {
            f.x = 5; // error
        }

        // Structs - Rule 1
        struct Foo
        {
            int x;
        }

        var foo = Foo();
        foo.x = 5; // changing values on a struct is acceptable

        IFoo ChangeData(IFoo f)
        {
            var foo = Foo();
            foo <- f;// copy command; will copy every value in the interface
            // same as:
            /*
                foo.x = f.x; //
            */
            return foo;
        }

        foo = ChangeData(default(IFoo)); // reassigning a struct is acceptable
        foo <- ChangeData(default(IFoo)); // copying values to a struct is acceptable

Reasoning

Interfaces are designed to show data, not modify it. Structs are for changing data.
